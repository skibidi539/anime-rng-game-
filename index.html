Tải Game

File game sẽ được cập nhật tại đây.

Cách mở game

  1. Cài Code::Blocks.
  2. Tải file game.
  3. Mở file .cpp hoặc .cbp.
  4. Nhấn F9 để chạy.

Cách chơi

  • Story: đánh quái kiếm vàng.
  • Hero Shop: mua nhân vật.
  • Hero Chest: mở rương.
  • Shard Exchange: đổi mảnh.
  • Upgrade: nâng cấp.
  • Redeem: nhập code nhận quà.
